# Peregrino

## Descripción

Peregrino es un videojuego de plataformas 2D desarrollado en HTML, CSS y JavaScript.

El jugador controla a un peregrino que debe recorrer distintos escenarios inspirados en Galicia, superar obstáculos, derrotar enemigos y enfrentarse a jefes finales para completar su viaje.

Actualmente el proyecto se encuentra en fase MVP (Minimum Viable Product).
